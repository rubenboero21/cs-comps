Useful Wireshark notes:

Packet capture (capture filters):
	To capture packets that involve a given IP. In capture filters bar:
		* host <IP address>

	To capture packets only sent from an IP:
		* src host <IP address>

	To capture packets only sent to an IP:
		* src host <IP address>

Display filter:
	To view only the ssh packets. In display filter bar:
		* ssh
